# Papago Traslator

먼저 만들어진 원조 앱을 사용하다 긴 글을 번역할 때 화면에 출력되지 않는게 불편해서 수정했습니다. 좋은 앱을 만들어주신 개발자님들께 감사 인사드립니다.

## 설치방법

<img width="2032" alt="5 download" src="https://user-images.githubusercontent.com/77876601/179506544-280e3e92-1b55-424c-bd7b-1424c917b3cc.png">
녹색 Code 버튼을 누르면 Download ZIP 버튼이 나타납니다. 눌러서 다운로드하고 압축해제 합니다.

<img width="822" alt="6 install" src="https://user-images.githubusercontent.com/77876601/179506551-368bfbcf-391e-4e33-8ff6-bbff6c6fe9cf.png">
터미널에서 압축해제된 폴더로 이동하고 아래 명령어를 입력하면 필요한 패키지를 다운받고 Raycast에 확장이 설치됩니다.

```bash
npm install && npm run dev
```

<img width="822" alt="7 run" src="https://user-images.githubusercontent.com/77876601/179506558-b74f6e96-f876-4bf1-849f-dbc8cc72c022.png">
위의 화면이 나타나면 설치가 끝났고 터미널을 종료해도 됩니다.

<img width="862" alt="8 init" src="https://user-images.githubusercontent.com/77876601/179506563-08929042-389e-46d2-b4c9-c450a5374a1c.png">
설치된 모습입니다.

<img width="862" alt="9 input key" src="https://user-images.githubusercontent.com/77876601/179506567-52a8c8a6-85cf-43f9-b4fe-bea14808327d.png">
파파고 API key를 입력합니다. key를 얻는 방법은 아래에 있습니다.

<img width="862" alt="10 randing" src="https://user-images.githubusercontent.com/77876601/179506571-b7fd21c2-0587-454c-8aa4-26c1f51e62e9.png">
<img width="862" alt="11 short" src="https://user-images.githubusercontent.com/77876601/179506579-1d8cb393-d91e-4260-b21e-8209c2e16a00.png">
